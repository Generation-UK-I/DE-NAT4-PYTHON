# Virtualisation and Containers in DE

This repository is intended to complement the Generation DE bootcamp; you will learn a wide range of technologies and use various different applications; however, each of them has different requirements, or work best in different environments.

We have three purposes:

1. Review virtualisation technologies, including virtual machines and containers.
2. Deploy a Linux virtual machine, connect to it and use it.
3. Understand, install, and use Docker to deploy containers.
